AI Auto Repair Advisor is a modern full-stack web application designed to help users diagnose vehicle issues, understand repair problems, and receive intelligent repair guidance using AI-powered assistance.

The project combines a fast frontend experience with a scalable backend architecture using modern JavaScript technologies.

# 📌 Project Overview

This application allows users to:

- Describe vehicle problems
- Receive AI-generated repair suggestions
- Understand possible causes of issues
- Improve troubleshooting decisions before visiting a mechanic

The goal is to reduce confusion, save time, and help users make informed repair decisions.
